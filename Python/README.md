# Examples
This is a collection of example code for accessing the U-M GPT Toolkit API service.  

**Required Parameters**   

Please create a .env in the same directory as the script you want to run with the following:

MODEL=X  
OPENAI_API_BASE=X   
OPENAI_API_KEY=X   

Change the model, base url, and api key to your parameters. 

Model options can be displayed by running the get_models.py file.
